Belajar HTML dan CSS

html digunakan untuk membuat tampilan website, css digunakan untuk mengatur tampilan website css

Materi HTML

  • h1
  • p untuk paragraf
  • img untuk gambar
  • table untuk table
  • input untuk form
  • li untuk list/daftar

Materi CSS

  1. margin, padding, podition, untuk mengatur jarak pada elemen html
  2. dispay untuk mengatur elemen html atau tidak tampil mendatar (kolom), atau menurun (baris)
  3. style untuk mengatur font
